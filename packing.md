<h2>🧳 行李準備清單</h2>
<p>澳門三天兩夜家庭旅遊，一月中旬出發</p>
</div>

<div class="content-section">
<h2>🌡️ 澳門一月天氣</h2>

<div class="highlight-box">
<ul>
<li>🌡️ <strong>氣溫：</strong>約 12°C - 19°C</li>
<li>☀️ <strong>天氣：</strong>乾燥、少雨，偶有寒流</li>
<li>⚠️ <strong>注意：</strong>早晚偏涼，中午較溫暖</li>
<li>💡 <strong>穿搭重點：</strong>洋蔥式穿法，方便穿脫！</li>
</ul>
</div>
</div>

<div class="content-section">
<h2>👕 穿衣建議</h2>

<h3>👨‍👩‍👧 大人穿搭</h3>

<div class="info-card">
<p><strong>上衣：</strong></p>
<ul>
<li>長袖襯衫、薄毛衣</li>
<li>衛生衣（怕冷可穿）</li>
</ul>

<p><strong>外套：</strong></p>
<ul>
<li>⭐ <strong>必備！</strong>厚外套或羽絨外套</li>
<li>早晚涼，一定要帶！</li>
</ul>

<p><strong>下身：</strong></p>
<ul>
<li>長褲、牛仔褲</li>
</ul>

<p><strong>鞋子：</strong></p>
<ul>
<li>👟 舒適好走的運動鞋/休閒鞋</li>
<li>⚠️ 要走很多路，不要穿新鞋！</li>
</ul>

<p><strong>配件：</strong></p>
<ul>
<li>薄圍巾（選用）</li>
<li>帽子（怕冷可帶）</li>
</ul>
</div>

<h3>👶 寶寶穿搭</h3>

<div class="info-card">
<p><strong>內層：</strong></p>
<ul>
<li>棉質包屁衣/衛生衣</li>
</ul>

<p><strong>中層：</strong></p>
<ul>
<li>長袖上衣、薄刷毛衣</li>
</ul>

<p><strong>外層：</strong></p>
<ul>
<li>⭐ 保暖外套、防風外套</li>
</ul>

<p><strong>下身：</strong></p>
<ul>
<li>長褲、刷毛褲</li>
</ul>

<p><strong>配件：</strong></p>
<ul>
<li>帽子、小襪子</li>
<li>薄毯子（推車/揹巾用）</li>
</ul>

<p>⚠️ <strong>注意：</strong>寶寶體溫調節能力較差，多帶一件備用！</p>
</div>

<h3>👗 旗袍體驗穿搭（Day 2）</h3>

<div class="info-card">
<ul>
<li>👙 裡面穿<strong>膚色/黑色打底衣褲</strong>（保暖又不影響外觀）</li>
<li>🧥 帶<strong>可脫的外套</strong>（拍照時脫掉）</li>
<li>👠 穿<strong>方便穿脫的鞋子</strong>（店家可能提供高跟鞋拍照）</li>
</ul>
</div>
</div>
